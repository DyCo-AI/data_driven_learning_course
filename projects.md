**Waymo Prediction Challenge.** Build models that predict multi-modal future trajectories of vehicles, pedestrians, and cyclists using recent motion history and map context over an 8-second horizon.

**Comma AI Controls Challenge.** Learn vehicle dynamics from real driving data and design controllers for steering, throttle, and braking to track a reference path.

**Offroad Driving: Modeling.** Develop predictive models for vehicle motion on unstructured terrain using camera, LiDAR, IMU, and proprioceptive data while accounting for slip, bumps, and elevation changes.

**Offroad Driving: Control Design.** Use terrain-informed data-driven models to design controllers, such as MPC or reinforcement learning, for autonomous off-road driving.

**Agile Quadrotor State Estimation.** Learn dynamics for high-speed quadrotor flight from vision and inertial data, then design estimators for pose, velocity, and orientation during aggressive maneuvers.

**Quadruped Locomotion Modeling.** Learn predictive models for contact-rich legged locomotion using proprioceptive and sensory data, including base motion, joint states, contact, and terrain effects.

For more details, refer to the **[project descriptions file <i class="fas fa-file-pdf"></i>](https://github.com/DyCo-AI/data_driven_learning_course/blob/main/PROJECTS/ddlc_project_ideas_spring2026.pdf)**.
