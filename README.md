# Data Driven Learning and Control for Dynamical Systems

This repository contains course materials, assignments, datasets, and code for
**Data Driven Learning and Control for Dynamical Systems** at Clemson University.

https://dyco-ai.github.io/data_driven_learning_course/

## Course Description

This course provides a rigorous introduction to contemporary methods for the
data-driven modeling, analysis, and control of dynamical systems. It unifies key
ideas from machine learning, optimization, control theory, and dynamical systems
to address modern challenges in modeling and controlling complex engineering
systems.

Topics include data-driven system identification, operator-theoretic
perspectives such as Koopman and Perron-Frobenius methods, reinforcement
learning for control, optimal and robust control methods, and computational
algorithms for high-dimensional systems. Throughout the course, theoretical
developments are complemented by algorithmic implementations and software-based
experimentation.

Applications are drawn from robotics, autonomous systems, power and energy
networks, fluid dynamics, and other areas where data and dynamics intersect. By
the end of the course, students will be prepared to formulate and solve
data-driven modeling and control problems using state-of-the-art tools and
methodologies.

## Student Learning Outcomes

Upon successful completion of this course, students will be able to:

- Develop and implement algorithms that leverage data for constructing
  reduced-order models of complex dynamical systems.
- Design and code data-driven modeling frameworks, including system
  identification and operator-theoretic methods.
- Formulate and analyze reduced-order models for prediction, stability
  assessment, and control design.
- Design data-driven control strategies using tools from machine learning,
  optimization, and control theory.
- Apply computational methods and software tools to real-world dynamical systems
  arising in robotics, autonomous systems, power networks, and fluid dynamics.

## Course Content

- **Weeks 1-2:** Data dimensionality reduction; clustering and classification
  algorithms; support vector machines; kernel methods.
- **Weeks 3-4:** Deep neural networks; backpropagation; stochastic gradient
  descent and other optimization methods for training.
- **Weeks 5-7:** Data-driven system identification; subspace identification
  methods; linear operator-theoretic approaches, including Koopman and
  Perron-Frobenius frameworks, for data-driven modeling.
- **Weeks 7-8:** Reduced-order modeling from data; empirical Gramians; balanced
  truncation; proper orthogonal decomposition.
- **Weeks 9-11:** Data-driven Lyapunov analysis and optimal control; model
  predictive control; model-free reinforcement learning and Q-learning.
- **Weeks 12-13:** Operator-theoretic methods for data-driven control, safety
  analysis, and invariant set computation.
- **Weeks 14-16:** Applications to robotics, autonomous systems, electric power
  networks, and fluid mechanics.

## Grading Policy

This course is project-intensive and emphasizes hands-on implementation of
data-driven modeling and control methods. There are **no quizzes and no midterm
exam**. Student performance will be evaluated according to the following
components:

- **Final Course Project: 40%**  
  A comprehensive open-ended project involving data-driven system
  identification, operator-theoretic modeling, optimal or reinforcement
  learning-based control, or other advanced topics discussed in the course.
  Deliverables include a written technical report, a code repository, and an
  in-class presentation or demonstration during the final week.

- **Homework Assignments: 25%**  
  Three to four assignments combining coding exercises, algorithm
  implementation, short analytical derivations, and data-driven experimentation.

- **Project Milestones / Progress Reports: 30%**  
  Two brief progress reports submitted during the semester to ensure continuous
  advancement on the final project. These may include preliminary results,
  system identification performance, or early control experiments.

- **Class Participation and Engagement: 5%**  
  Active engagement in discussions, participation in peer feedback sessions,
  involvement in coding labs, and contributions to the collaborative learning
  environment.
